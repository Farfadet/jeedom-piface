== PiFace

=== Description
Le plugin PiFace permet de piloter une carte PiFace connecté a votre raspberry  et d'afficher sur les informations sur le  Jeedom de votre choix. Pour cela 2 modes sont disponibles, le mode local et le mode réseau.
